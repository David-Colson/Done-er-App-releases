# Done-er Daily Report — installers

Windows installers for **Daily Report**, a local, private standup / bullet-journal tracker.
Your data never leaves your machine — everything lives in a local database under your own
user profile; the only network access the app ever makes is checking this page for updates.

## Install

1. Grab the newest `daily-report-setup-<version>.exe` from
   [Releases](https://github.com/David-Colson/Done-er-App-releases/releases/latest).
2. Run it. Windows SmartScreen will warn because the installer isn't code-signed —
   click **More info → Run anyway**. It installs per-user (no admin prompt) and launches
   itself when done.
3. That's it. The app updates itself automatically when a new version is published here
   (downloaded in the background, installed the next time you close the app).
